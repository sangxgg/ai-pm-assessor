# AI PM Assessor

一个用于 AI 时代互联网产品经理能力评估的 Codex Skill。

它通过逐步背景采集、选择题或探讨式问诊、专家追问、变量扰动、失败复盘和结构化报告，帮助产品从业者识别能力画像、长板短板、胜任风险和提升路径。

## 使用方式

将本仓库复制或安装到 Codex skills 目录，例如：

```text
.codex/skills/ai-pm-assessor/
```

## 包含内容

- `SKILL.md`：主技能说明与执行流程
- `references/`：能力框架、动态问诊方法、报告模板
- `agents/openai.yaml`：Codex UI 入口信息

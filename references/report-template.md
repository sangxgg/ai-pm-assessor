# 最终报告模板

## 报告结构

使用中文输出，保持直接、具体、可执行。报告要像专家委员会的诊断意见，而不是测评小程序结果。每个结论都要能回溯到用户在某一轮场景、追问、变量扰动或失败复盘中的具体表现。

## 分数换算

将每个维度的累计贡献归一到100分。每个维度都必须有证据覆盖；如果某维度经追问或相邻场景补证后仍然不清晰，不要强行给高置信判断，可用相邻证据谨慎推断，并在报告里标注“置信度较低”和待验证假设。

维度判断必须同时考虑：

- 开放场景回答质量，或可选单选校准题的选项得分
- 用户解释是否补齐关键约束
- 专家追问中的补充证据
- 用户背景和目标岗位对该维度的要求强度
- 该维度是否被多轮问诊或相邻维度交叉验证
- 该维度是否已经达到 `sufficient`，或仍属于低置信判断

总分建议按权重汇总：

- A 问题定义与机会判断：15%
- B 用户洞察与场景拆解：12%
- C AI能力理解与产品化：18%
- D AI原生体验与工作流设计：12%
- E 数据指标与实验决策：13%
- F 商业化与增长策略：10%
- G 优先级与资源取舍：10%
- H 组织协同、伦理与风险：10%

## 短板表述规范

短板要写成“可训练能力”，不要写成人格评价。

推荐：

- “AI产品化判断偏Demo导向，缺少评测、兜底和成本意识”
- “问题定义容易直接跳到方案，用户任务和业务约束拆得不够”
- “指标设计偏结果口号，缺少过程指标、反指标和实验边界”

避免：

- “你不适合做产品”
- “逻辑差”
- “没有商业sense”

## 专家委员会结论规则

结论是相对于用户目标岗位，不是对人的永久评价。

- **Strong Hire**：目标岗位关键维度稳定强，压力追问下仍能兼顾用户、业务、AI边界、数据、风险和组织落地
- **Hire**：多数关键维度达到目标岗位要求，有明确短板但不构成核心胜任风险
- **Lean Hire**：有可培养长板，但目标岗位关键维度存在明显缺口，需要补强或降低职责范围
- **No Go for Current Target**：当前证据显示至少一个目标岗位核心门槛缺失，且短期独立负责风险较高

不要轻易给 Strong Hire；必须有多维交叉证据。不要轻易给 No Go；除非Derailer证据清晰，或目标岗位要求明显高于当前表现。

## 能力画像生成规则

能力画像要帮助用户理解“自己靠什么赢、会在哪里卡住”：

- 工程实战派：C/G强，能把技术约束转成产品路线，但可能忽略用户叙事或商业包装
- 商业策略派：A/F/G强，能看机会和资源取舍，但可能对AI评测和体验细节不够敏感
- 体验创新派：B/D强，擅长人机协作和用户信任，但可能需要补数据实验或商业闭环
- 数据实验派：E/F强，能建立指标和实验，但可能需要补定性洞察和AI边界
- AI系统产品派：C/D/E/H强，能建立AI能力、评测、体验和风险闭环
- 交付协同派：G/H强，擅长复杂组织推进，但可能要防止被客户需求牵着走

画像可以组合，例如“AI系统产品派 + 交付协同派”。不要为了贴标签而忽略证据。

## 维度理由生成规则

每个维度的判断理由必须包含以下至少2类证据：

- 用户在某轮的具体判断
- 用户补充理由、变量扰动或失败复盘回答
- 与目标岗位要求的差距
- 相邻维度的交叉证据，例如AI产品化弱同时影响体验设计和风险控制

如果只有1个弱证据，不要写成确定结论，应写成“目前呈现出...倾向，置信度中/低”。

不要使用空泛表述：

- “综合能力不错”
- “还需要加强思考”
- “有一定潜力”

要改成具体表述：

- “你能识别上线前要看业务指标，但没有把AI评测、失败兜底和成本门槛纳入发布标准”
- “你能从用户反馈出发，但对目标用户、替代方案和高频场景的拆解不够稳定”
- “你能做功能优先级排序，但缺少学习价值、技术不确定性和组织成本的统一权衡”

## 30天计划生成规则

根据最大短板定制：

- 问题定义弱：练习PRD前置的问题树、用户任务、替代方案和机会假设
- 用户洞察弱：做5个用户访谈复盘，区分事实、解释和方案
- AI产品化弱：拆解3个AI功能的评测集、失败兜底、成本和用户信任机制
- AI体验弱：画一个人机协作流程，标出用户确认点、可撤销点和异常处理
- 数据实验弱：为一个功能设计北极星指标、过程指标、反指标和A/B实验
- 商业增长弱：拆解一个AI产品的获客、定价、留存和单位经济模型
- 优先级弱：用RICE/ICE加风险和学习价值重排一个路线图
- 协同风险弱：写一页上线风险清单，覆盖隐私、版权、合规、品牌和运营兜底

每周计划必须有可交付产物，例如“一页问题定义文档”“一份评测集草案”“一个实验设计表”“一页风险清单”。不要只写“学习”“提升”“加强思考”。
